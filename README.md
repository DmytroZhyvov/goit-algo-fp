## Завдання 7. Використання методу Монте-Карло

---

## Опис завдання

У цьому завданні було реалізовано алгоритм методу **Монте-Карло** для моделювання великої кількості кидків двох ігрових кубиків.  
Метою було визначити **ймовірності появи кожної можливої суми** чисел (від 2 до 12) та порівняти отримані результати з **аналітичними значеннями**.

---

## Метод Монте-Карло (ідея)

1. Два гральні кубики кидаються велику кількість разів.
2. Для кожного кидка обчислюється сума значень на кубиках.
3. Підраховується, скільки разів кожна сума з’являється.
4. Ймовірність кожної суми обчислюється як: probability = count / total_throws

Чим більша кількість симуляцій, тим ближчими є результати до теоретичних.

---

## Порівняння з аналітичними розрахунками

Аналітичні ймовірності для двох кубиків відомі та залежать від кількості можливих комбінацій (усього 36).  
Результати, отримані методом Монте-Карло, **дуже близькі до теоретичних значень**, особливо для суми 7, яка має найбільшу кількість комбінацій.

---

## Висновки

- Метод Монте-Карло **коректно моделює** процес кидання двох кубиків.
- Отримані ймовірності **збігаються з аналітичними значеннями** в межах малої похибки.
- Чим більша кількість симуляцій, тим **меншою стає похибка** результатів.
- Метод Монте-Карло є простим у реалізації та добре підходить для задач, де аналітичні розрахунки складні або неможливі.
- Для даної задачі метод підтвердив свою **правильність та надійність**.

---

**Висновок:**  
Метод Монте-Карло є ефективним інструментом для статистичного моделювання та дозволяє з високою точністю відтворювати теоретичні ймовірності при достатній кількості симуляцій.
